Cross Lingual Information Retrieval

Our Cross-Lingual Information Retrieval tool is designed to enhance document retrieval across multiple languages without the need for direct translation. This project uses advanced neural network architectures, specifically Bidirectional LSTM, along with sophisticated nearest-neighbor algorithms (Annoy and HNSWLib), to retrieve the most relevant documents from a vast corpus based on user queries in any of the seven languages including Arabic, Bengali, and Japanese, among others. Our model utilizes the XOR-TyDi dataset for training and testing, comprising diverse queries and documents that challenge the model with real-world complexities and variability. This dataset supports our tool in learning nuanced language patterns, thereby improving its effectiveness and robustness in practical applications. The implementation tests different nearest-neighbor search techniques, comparing their performance to optimize retrieval accuracy and efficiency. The exploratory analysis of these models reveals significant insights into the trade-offs between speed and precision in retrieval tasks. Future work will focus on integrating BERT-based tokenization for improved accuracy and exploring exact nearest neighbor search metrics like cosine similarity to refine the search process further. For more insights and source code, please visit our GitHub repository: Cross-Lingual Information Retrieval GitHub by cicking on details below.

Moodd Based Media Recommendation

This project, inspired by the success of platforms like Spotify, aims to enhance music discovery and personalize user experiences through an Advanced Music Genre Classification System. Developed by Sanjeeth Nagappa Chakkrasali, Shaun Noronha, and Khushita Joshi, the system uses sophisticated audio analysis and machine learning models, including mood recognition technology, to offer personalized music recommendations based on the user's current emotional state. Utilizing the GTZAN Dataset, the system categorizes 1000 audio tracks across 10 genres, analyzing features such as Chroma STFT, RMS Mean, and MFCCs. Our exploratory data analysis includes a BPM boxplot and correlation heatmap to refine genre classification accuracy. We implemented several models, including Sequential Dense Models and Enhanced Dense Models with dropout variations, achieving up to 94.03% validation accuracy. The mood detection component uses convolutional neural networks to recognize emotional states from images, influencing music recommendations tailored to the listener's mood. The recommendation engine employs cosine similarity to suggest tracks with similar musical features, enhancing the music discovery experience.

DocQueryAI

DocQueryAI revolutionizes information retrieval from PDF documents through its advanced Retrieval-Augmented Generation (RAG) technology. Designed to function as an intelligent chatbot, DocQueryAI leverages the LLaMA index and OpenAI's API, enabling it to perform sophisticated document querying and provide precise, contextually relevant answers. Key features of DocQueryAI include its ability to be trained on custom PDFs or other document formats, making it highly adaptable to various user needs. It also features an interactive query engine that allows users to ask questions and receive detailed responses based directly on the document's content. Additionally, the tool offers efficient index persistence, enabling the saving and reloading of indexes to optimize both time and computational resources. Getting started involves cloning the repository, setting up a virtual environment, installing necessary dependencies, and running a Jupyter notebook for interactive development and testing. This tool uses technologies such as the LLaMA Index for document indexing and the OpenAI API for generating responses, combined with the flexibility of Jupyter Notebooks and the Python programming language for implementation. For further information, including installation instructions and source code, please visit the GitHub repository by clicking on details below.
